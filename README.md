# Микрокапитализм: Жизнь на 1 доллар

## 🎮 Production-ready Telegram бот для экономической игры

### Структура
```
doll/
├── bot.py           # Точка входа
├── config.py        # Конфигурация
├── db.py            # База данных (SQLite/PostgreSQL)
├── handlers/         # Обработчики команд
│   ├── __init__.py
│   └── main.py
└── requirements.txt
```

### 🚀 Запуск

```bash
pip install -r requirements.txt
python bot.py
```

### 📊 Экономика

**Ресурсы:**
- 🍎 Еда (базовая цена: $10)
- 🔧 Материалы ($20)
- ⚡ Энергия ($15)

**Бизнесы:**

- 🌾 Ферма ($20) → еда

- ⚡ Электростанция ($40) → энергия

- Динамические цены на рынке (спрос/предложение)
- Переводы между игроками (2% комиссия)
- VIP за Stars (+20% к производству)
- Реферальная система (+$5 за друга)
- Работа в чатах (/отправить, /топ, /рынок)

### 📦 Технологии

- Python 3.10+
