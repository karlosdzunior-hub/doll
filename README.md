# Микрокапитализм: Жизнь на 1 доллар

Telegram-бот для игры в стиле "clicker" с механикой построения бизнеса.

## 🚀 Запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите токен бота:
```bash
export BOT_TOKEN="your_bot_token_here"
```

3. Запустите бота:
```bash
python bot.py
```

## 📁 Структура проекта

- `bot.py` - Основной файл бота с обработчиками
- `db.py` - Работа с базой данных SQLite
- `config.py` - Конфигурация игры
- `utils.py` - Игровая логика и утилиты

## 🎮 Возможности (MVP)

### Меню
- 🏠 Мой баланс - просмотр баланса и бизнесов
- 💼 Бизнес - создание и апгрейд бизнесов
- 🎲 События дня - случайные события (пожар, налог, бонус, находка)
- ⭐ VIP/Покупки - покупка VIP, щитов, лотерея
- 🔝 Лидерборд - топ игроков
- 🔁 Рефералы - приглашение друзей

### Бизнесы (MVP)
- 🍋 Лимонадная ($1) - базовый доход $1
- 🌾 Ферма ($5) - базовый доход $2

### События
- 🔥 Пожар - урон от $1 до $5
- 📋 Налог - урон от $1 до $3
- 🎁 Бонус - награда от $2 до $10
- 💰 Находка - награда от $1 до $5

### VIP
- Стоимость: 5 Stars
- Длительность: 7 дней
- Бонус: $5 ежедневно
- Защита от негативных событий

## 📝 Расширение

### Добавление нового бизнеса
В `config.py` добавить в `BUSINESS_TYPES`:
```python
"shop": {
    "name": "Магазин 🏪",
    "base_income": 3.0,
    "base_cost": 10.0,
    "upgrade_cost_multiplier": 1.6
}
```

### Добавление нового события
В `config.py` добавить в `EVENTS`:
```python
"earthquake": {
    "name": "Землетрясение 🌋",
    "min_damage": 5,
    "max_damage": 15
}
```
